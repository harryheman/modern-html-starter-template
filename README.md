## Installation

```
git clone https://github.com/harryheman/Modern-HTML-Starter-Template.git
cd Modern-HTML-Starter-Template
npm i / yarn
```

## Usage

### Development Server

```bash
npm run dev / yarn dev
```

### Production Server

```bash
npm start / yarn start
```
